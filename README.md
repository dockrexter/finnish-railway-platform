# Finnish Railway Platform

End-to-end data engineering project using Finnish railway data to practice modern data platform patterns.

- Ingestion of public railway data (API/files) into a lakehouse-style structure
- Streaming concepts for near real-time updates
- ETL/ELT pipelines with PySpark/SQL
- Slowly Changing Dimensions (SCD) implementation for tracking history
- Focus on Databricks-style architecture and best practices
